# 🏠 GharBasao
GharBasao is a full-stack MERN real estate web application where users can buy, rent, and sell properties. 
Users can browse listings, save favorites, contact property owners, and schedule property visits.

## 🚀 Features
- User Authentication (Email & Google Sign-In)
- Create, Update, and Delete Property Listings
- Buy & Rent Property Listings
- Wishlist (Save Favorite Properties)
- Contact Property Owner
- Schedule Property Visits
- Owner Accept/Reject Visit Requests
- Email Notifications
- Property Location with Interactive Map
- Fully Responsive Design

## 🛠️ Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS
- Firebase Authentication
- Leaflet Maps

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Brevo Email API

## ⚙️ Installation
### Clone the repository
```bash
git clone https://github.com/AsifShaik-786/GharBasao.git
```
### Install dependencies
#### Backend
```bash
cd api
npm install
npm run dev
```
#### Frontend
```bash
cd client
npm install
npm run dev
```
